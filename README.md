# WebTodo

> A simple and modern ASP.NET Core MVC web application for task management, demonstrating CRUD operations, clean architecture, and responsive user interfaces.

## 📚 About the Project

**WebTodo** is a sample To-Do List application built with **ASP.NET Core MVC**. It demonstrates how to create a complete web application where users can manage their daily tasks through an intuitive interface.

The project focuses on clean code, maintainability, and modern ASP.NET Core development practices, making it an excellent learning resource for developers studying MVC applications.

This repository is intended for educational purposes and serves as a practical example of full-stack web development using ASP.NET Core.

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/LuizErnica/WebTodo.git
```

### Navigate to the project

```bash
cd WebTodo
```

### Restore dependencies

```bash
dotnet restore
```

### Apply database migrations

```bash
dotnet ef database update
```

### Run the application

```bash
dotnet run
```

Open your browser and navigate to:

```
https://localhost:xxxx
```
